# Sender::send 文档翻译

```rust
/// 尝试向所有活跃的 [`Receiver`] 句柄发送一个值，
/// 如果无法发送则将其返回。
///
/// 当存在至少一个活跃的 [`Receiver`] 句柄时，发送成功。
/// 不成功的发送是指所有关联的 [`Receiver`] 句柄都已被丢弃的情况。
///
/// # 返回值
///
/// 成功时，返回已订阅的 [`Receiver`] 句柄数量。
/// 这并不意味着有这个数量的接收者会看到该消息，因为接收者可能在
/// 收到消息之前丢弃或滞后（参见[滞后](self#lagging)）。
///
/// # 注意
///
/// 返回 `Ok` **并不** 意味着发送的值会被所有或任何活跃的
/// [`Receiver`] 句柄观察到。[`Receiver`] 句柄可能在收到消息之前被丢弃。
///
/// 返回 `Err` **并不** 意味着后续对 `send` 的调用会失败。
/// 可以通过调用 [`subscribe`] 创建新的 [`Receiver`] 句柄。
```
